# crawling
#### 목표 ####: 유튜브는 유행이 즉각적으로 드러나고, 유튜브의 댓글에서 시청자의 반응이나 감정이 직접적으로 드러나는 편이다. 정제되지않고 솔직한 raw data들이 많은 유튜브를 크롤링하고 싶었다. 본 프로젝트는 유튜브에서 인기가 많은 youtube 먹방 크롤링 후, 어떤 음식의 빈도수가 높은지 시각화하여 유행분석 및 인기음식을 알아보았다.
